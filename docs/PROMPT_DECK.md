# Hermes Bot - Prompt Deck

Короткие фразы для запуска работы с телефона. Пиши естественно; MAIN сам разберёт, как маршрутизировать задачу.

## Утром

- **"новые задачи: <список>"** -> MAIN классифицирует и распределяет.
- **"продумай фичу: <идея>"** -> Product Manager готовит spec.
- **"исследуй: <тема>"** -> Research / Analysis готовит memo.

## Статус

- **"что у меня?"** -> краткий статус по `ROADMAP.md`.
- **"что в работе?"** -> только активные задачи.
- **"что висит?"** -> открытые вопросы и блокеры.

## Баг

- **"баг: <скриншот + текст>"** -> QA / BugFix оформляет баг, воспроизводит и готовит fix.
- **"сломалось после <действие>"** -> QA ищет регрессию и пишет план проверки.

## Деплой

- **"deploy"** -> MAIN проверяет production lock и готовит безопасный deploy.
- **"откат прода"** -> MAIN останавливается на irreversible action и эскалирует решение.

## Ответы на эскалации

- **"да, делай вариант A"** -> MAIN снимает блокировку и продолжает.
- **"нет, вариант B"** -> MAIN обновляет план и продолжает с выбранным вариантом.
- **"стоп, передумал"** -> MAIN отменяет задачу и обновляет `ROADMAP.md`.
