# Hermes Bot - Mobile Workflow

Инструкция для АЮ: как запускать и принимать работу с iPhone без технических деталей.

## Запустить новую задачу

1. Открой Codex на iPhone.
2. Выбери проект `ceo-aivocado/hermes-agent`.
3. Открой chat `Hermes Bot MAIN / Orchestrator`.
4. Напиши коротко, что нужно сделать. Примеры есть в `docs/PROMPT_DECK.md`.
5. Если есть скриншот, приложи его сразу.
6. Отправь сообщение и не расписывай технический план; MAIN сам разложит задачу.

## Проверить статус

1. Открой `Hermes Bot MAIN / Orchestrator`.
2. Напиши: `что у меня?`
3. MAIN прочитает `ROADMAP.md` и ответит коротким списком.

## Ответить на вопрос от Codex

1. Если пришёл Telegram push, открой ссылку из сообщения.
2. Ответь прямо в MAIN chat: `да, делай вариант A` или `нет, вариант B`.
3. Если надо остановить задачу, напиши: `стоп, передумал`.

## Сообщить баг

1. Открой MAIN chat.
2. Напиши: `баг: <что случилось>`.
3. Приложи скриншот или короткое видео.
4. Укажи, что ожидал увидеть и что увидел вместо этого.

## Что не нужно делать

- Не пиши команды терминала.
- Не выбирай ветки, labels или PR вручную, если MAIN не попросил.
- Не присылай секреты в chat. Секреты добавляются только через Codex Cloud secrets.
